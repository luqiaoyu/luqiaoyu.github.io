Python

注：这里记的默认针对Python3。

  1. a = (1,)：a是一个tuple，长度为1，即len(a) = 1。,的存在是为了表示这是一个tuple，与(1)表示优先级相区分。
  2. Python的传参方式是引用传递。所有的Python变量（variable）都是一个引用（reference），指向真正的对象（object）。
  3. Java，C++中的?:三元运算符，在Python中不存在，Python中可以用if else等价实现，例如：a = 0 if flag == True else 1
  4. Immutable sequences: tuple, str, bytes
  5. Set elements must be hashable. The set type is not hashable, but frozenset is, so you can have frozenset elements inside a set.

  6. list的reverse方法和[::-1]的区别：

    reverse reverses the list in-place, see the manual, while [::-1] gives a new list in reversed order.

    Note that if you don’t actually need a list with the values in the reverse order (just an iterable), using the reversed builtin function can be more efficient. Something like for value in reversed(p) doesn’t add a significant amount of overhead (when compared with any other iteration). p[::-1] and p.reverse() are both O(N), but reversed(p) is O(1). Of course iterating on it is O(N), but so is iterating on p without reversing.

  7. 关于dict：

    d[k] raises an error when k is not an existing key

    d.get(k, default) returns default when k is not an existing key

    d.setdefault(k, default) sets d[k] = default and returns default when k is not an existing key

    d = collections.defaultdict(callable) sets d[k] = callable() and returns d[k] when k is not an existing key, e.g. callable = list

    subclass dict or other mapping types: the __missing__ method
