  1. Introduction

    主要介绍了这本书的受众，这本书组织内容和组织方式的初衷，用于教学的建议和相关资料。不太重要。

    Keys:

    • Java从C/C++继承而来
    • The introductory portion ends with the Error Handling with Exceptions chapter, but you may also want to supplement this with an introduction to JDBC, Servlets and JSPs.
    • [Coding Standards] Because Java is a free-form programming language, you can continue to use whatever style you’re comfortable with.
    • This book focuses on and is tested with Java SE5/6.

    Others:

    • experiment and iteration is a technique that works well in both presentation and program design
    • introductory seminar: www.MindView.net
    • To take an example from C, if you memorize the operator precedence table (I never did), you can write clever code. But if you need to think about it, it will also confuse the reader/maintainer of that code. So forget about precedence, and use parentheses when things aren’t clear.
